# Changelog

All notable changes to Flowby will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup
- Core language features (v5.1)
  - Python-style indentation blocks
  - Variables and constants
  - Control flow (if/else, when, for, while, break, continue)
  - Functions with parameters and return values
  - Module system (library, import, export)
  - Web automation (navigate, click, type, wait, assert)
  - OpenAPI resource integration (5 phases)
  - Input expression with defaults and type conversion
  - Expression statements
- Test suite
  - Unit tests for lexer, parser, interpreter
  - Integration tests
  - Grammar alignment tests
- Documentation
  - Language reference (MASTER.md)
  - Examples
  - API documentation

### Changed
- Project rebranded from "ADS Registration System DSL" to "Flowby"
- File extension standardized to `.flow`

### Fixed
- None yet
